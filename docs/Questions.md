# Terra In-cognita: 기획 구체화를 위한 질문 리스트

현재 개발 진행을 위해 구체화가 필요한 부분들을 정리했습니다. 답변해주시면 반영하여 개발하겠습니다.

## 1. 월드맵 & 이동 시스템
* **맵의 크기와 형태**: 
    * 고정된 크기(예: 20x20 그리드)인가요, 아니면 무한히 확장되는 오픈 월드 방식인가요?
    => 오픈월드 방식. 기본은 지구를 기반으로 유사하게 제작. 영토 전쟁 및 1만~10만 유저를 수용할 수 있을 크기가 필요하다.
    맵 구성은 월드맵과 각 타일에 해당하는 2차 맵으로 구성된다. 더 하위 레이어 맵이 존재할 수도 있다.
    * 이동 방식은 턴제(한 칸씩 이동)인가요, 아니면 실시간 이동인가요?
    => 실시간 이동이나 거리에 따라 지형에 따라 이동속도가 달라진다.

    * **이동 비용**: 이동 시 에너지나 시간이 소모되나요?
    => 기본적으로 인간은 스태미나가 존재하고 기계류는 배터리나 에너지가 존재한다.
    

## 2. 건설 & 자원 시스템
* **건설 가능한 시설**: 
    * 초기 단계에서 건설 가능한 시설 종류는 무엇인가요? (예: 에너지 발전소, 광산, 주거지 등)
    => 다른 게임과 비슷한 시스템으로 구성한다. 창고와 채집시설(나무,광물, 식량 등)
    * 건설 제한 조건이 있나요? (예: 특정 지형에서만 건설 가능, 영토 소유권 필요 여부)
    => 기본시스템은 제한없으나 상위 건물로 갈수록 자원양이 많아지거나 기술이 요구된다.
* **채집 방식**:
    * 자원 타일 위에 서 있으면 자동으로 채집되나요, 아니면 '채집' 버튼을 눌러 액션을 수행해야 하나요?
    => 액션기반이다. 기본적으로 AI를 통해 자율적으로 채집하고 휴식을 취하고, 창고가 가득차면 시장을 통해 매매한다.
    * 자원은 무한한가요, 아니면 타일마다 매장량이 정해져 있나요?
    => 필수재는 무한하거나 양이 많고 일정시간마다 생성된다. 희귀자원은 타일마다 정해진 양이 있으며 일정시간마다 생성된다. 밸런스 핵심은 초보자에게 위기를 주지 않고, 고급 자원이나 기술로 갈수록 한정된 자원과 기술이 요구된다.

## 3. NPC & 도시 - 초기 구현
* **주요 도시(NPC)**:
    * 초기에 등장할 주요 국가나 도시의 이름과 특징(컨셉)을 2~3개 알려주세요.
    => 한걱의 서울, 미국의 뉴욕, 중국의 상해, 일본의 도쿄, 유럽의 도시들
    세계적으로, 지역별로 유명한 국가의 대도시를 다 반영하자. 실제 경제지표를 참고하면 좋다.
    * 도시 진입 시 가능한 상호작용은 무엇인가요? (예: 상점 이용, 퀘스트 수주, 회복 등)
    => 직급에 따라 다양하며 일반유저는 경제활동이나 상점 이용, 퀘스트 수주, 회복 등이 가능하다. 전투 리그참여 등. 정부직책을 맡거나 독재자, 국회의원이 되면 더 다양한 역할을 위해 기능이 추가된다.

## 4. 경제 & 화폐
* **초기 화폐**:
    * 게임 시작 시 지급되는 초기 자금과 자원량은 어느 정도가 적당할까요?
    => 유저 쏠림현상을 막기위해 유저수가 적은 국가는 더 많은 지원을 해준다.
    * 자원(에너지/광물)과 화폐(크레딧)의 대략적인 교환 비율이나 가치 개념이 있나요?
    => 자유경제시장을 기반으로 하나 하이퍼인플레이션이나 밸런스 유지를 위해 무한한것은 없고 희소성을 부여해야한다.

## 5. 기타
* **게임 오버/승리 조건**: 현재 단계에서 고려해야 할 엔딩 조건이 있나요?
=> 엔딩은 존재하지 않으나 한유저가 지구 전체를 독점, 통일하여도 작은단위에서 균열이 생겨 분열되는 시스템을 구현해야한다.
