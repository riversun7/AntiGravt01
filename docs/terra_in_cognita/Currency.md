# 화폐

## 개론
- 게임 내 경제의 핵심
- 인플레이션 제어 중요

## 화폐의 종류
- **Gold (골드)**: 가장 기초적인 화폐.
    - 획득: 사냥, 퀘스트, 상점 판매
    - 소모: 아이템 구매, 강화, 수리
- **Gem (잼)**: 유료 재화.
    - 획득: 결제, 업적 보상
    - 소모: 가속, 편의성 기능, 스킨

## 경제 구조
- 주식 시장 모티브
- 기업 ⇒ 주식 << 화폐(투자자)
- 국가(중앙은행) ⇒ 화폐, 채권 <<

## 4. 인플레이션 방지책
- 소모처 다각화 (Sinks)
    - 수수료 (경매장, 거래소)
    - 강화 파괴 및 복구
    - 영지 유지비 (세금)
- 생산량 조절 (Sources)
    - 피로도 시스템 (하루 획득량 제한)
    - 드랍율 조정
